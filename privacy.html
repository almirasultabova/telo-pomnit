4. Как мы храним и защищаем данные

Где хранятся данные

Все персональные данные хранятся исключительно на серверах, расположенных на территории Российской Федерации (хостинг Beget, Москва). Обработка персональных данных за пределами России не осуществляется.

Защита

  • Передача данных — только по зашифрованному протоколу HTTPS
  • Авторизация — через подпись Telegram (initData), токены JWT с ограниченным сроком действия
  • База данных доступна только с сервера, не открыта наружу
  • Каждая участница видит только свои данные
  • Ежедневное резервное копирование базы данных

Передача третьим лицам

Ваши данные не продаются и не передаются третьим лицам, кроме технических сервисов, обеспечивающих работу программы:

  • Telegram — авторизация и уведомления
  • ЮКасса — обработка платежей (передаётся только email для чека)
  • OpenAI — обработка сообщений AI-ассистента (данные обезличены в рамках сессии)
  • Яндекс — отправка email через SMTP

5. Срок хранения данных

Данные хранятся в течение всего времени вашего участия в программе и после её завершения — для обеспечения вам доступа к личной истории наблюдений. Доступ к записям сохраняется навсегда после окончания потока.

При удалении аккаунта (право на забвение) данные обезличиваются немедленно и физически удаляются в течение 30 дней.

В соответствии с 152-ФЗ вы имеете право:

  • Получить свои данные — выгрузить все данные в формате JSON или PDF через раздел «Мой поток» в приложении
  • Исправить данные — обратиться по контактам ниже
  • Удалить данные — запросить полное удаление аккаунта и всех данных через приложение или по email

При первом открытии Telegram Mini App вы даёте явное согласие на обработку персональных данных, включая данные о состоянии здоровья (записи дневника тела, чекины, стоп-реакции). Без согласия использование приложения невозможно.

Согласие фиксируется в базе данных с указанием даты и времени.

Сайт telo-pomnit.ru не использует файлы cookie для отслеживания пользователей. Telegram Mini App использует localStorage браузера исключительно для хранения токена авторизации на вашем устройстве.
