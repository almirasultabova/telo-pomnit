# Тело помнит — описание проекта

## Что это за проект

**«Тело помнит»** — сайт интегративной терапевтической группы.

Психосоматика + телесная терапия + внедрение в реальную жизнь.
5 недель, 9 живых встреч, 2 ведущие, Zoom + закрытый Telegram-чат.

Цель сайта: конвертировать посетителя в заявку через Telegram —
кнопки «Забронировать место» и «Задать вопрос» ведут к ведущим напрямую.

## Страницы проекта

### Существующие

#### `landing.html` — Лендинг v1 (яркий, многосекционный)
- **Стиль:** светлые анимированные блобы (лаванда, персик, розовый), SVG-иллюстрация тела
- **Секции:** hero → боль → что это → отличие от рынка → метод → формат → ведущие → для кого → отзывы → 3 тарифа → FAQ (7 вопросов) → CTA
- **Тарифы:** 3 варианта (7 900 ₽ × 2 + 12 000 ₽)
- **Акцент:** полнота информации, закрытие всех возражений через FAQ
- **Копирайтинг:** широкий охват болей и инструментов метода

#### `landing2.html` — Лендинг v2 (editorial, минимализм)
- **Стиль:** тёплый крем + глубокий лесной зелёный, дышащий круг в hero
- **Секции:** hero → боль → что меняется → процесс по неделям → формат встреч → кейс Екатерины → 1 тариф → disclaimer → финальный CTA
- **Тарифы:** 1 вариант (12 000 ₽ — «Полная перезагрузка»)
- **Акцент:** маршрут по неделям, конкретный пример результата, минимум слов
- **Копирайтинг:** «стратегия выживания», прямая речь, фокус на теле

> **Статус:** оба лендинга в работе, финальный выбор основного — после сравнения.

---

### Планируются

#### `thanks.html` — Страница «Спасибо»
Показывается после того, как человек оставил заявку или записался.
Подтверждает действие, снижает тревогу, даёт следующий шаг.

*Примерное содержание:*
- подтверждение («Вы сделали шаг»)
- что произойдёт дальше (когда напишет ведущая)
- ссылка на Telegram-канал или чат для ожидающих

#### `payment.html` — Страница оплаты / тарифов
Отдельная страница с деталями тарифа и способом оплаты.
Может включать форму или кнопку платёжной системы.

*Примерное содержание:*
- тариф «Полная перезагрузка» — 12 000 ₽/мес
- что входит (список)
- реквизиты или кнопка оплаты (Tinkoff, ЮКасса и др.)
- контакт для подтверждения

## Дизайн-система

### Общее
- Все файлы — самодостаточные HTML (без фреймворков, без CDN)
- Шрифты — системные (`Georgia` для заголовков, `-apple-system` для текста)
- Иконки — инлайн SVG
- Анимации — CSS `@keyframes` + JS `IntersectionObserver` для reveal

### landing.html
| Параметр | Значение |
|---|---|
| Фон | `#fdfaff` + 4 анимированных CSS-блоба |
| Основной цвет | `#7c3aed` (фиолетовый) |
| Акцент | `#c026d3` (малиновый) |
| Текст | `#1a0530` |
| Карточки | `rgba(255,255,255,0.7)` + backdrop-filter blur |
| Анимация фона | Плавающие блобы (CSS keyframes, 20–30s) |
| Иллюстрация | SVG-силуэт тела с пульсирующими кольцами |

### landing2.html
| Параметр | Значение |
|---|---|
| Фон | `#f7f4ef` (тёплый крем) |
| Основной цвет | `#2a4a38` (лесной зелёный) |
